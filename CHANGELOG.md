## [0.3.0] - 2026-04-14

### 新增
- Web 与 CLI 统一展示项目元数据，包括版本、作者、许可证、更新方式与源码来源占位信息
- 远程 Linux 烟测脚本新增 SSH 私钥、SSH 选项与真实服务健康检查能力
- 发布文档新增外部 Linux / Windows 测试说明，其中 Windows 可使用虚拟机作为验证目标

### 变更
- README、发布流程和外部测试文档统一改为面向公开发布的表达方式
- 贡献指南去掉仓库所有者占位符，改为适用于正式仓库发布前后的通用说明
- `rich` 依赖上限收紧到 `<14`，降低与现有 `gemini-cli` 环境共装时的冲突概率

### 修复
- 移除日志中的占位发布日期，避免对外形成错误版本时间线
