# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.3.0] - 2024-01-XX

### 新增
- 🎨 全新现代化 CLI 界面，基于 rich 库实现
  - 彩色输出和交互式菜单
  - 类似 Claude Code 的现代感设计
  - 表格、面板、进度条等丰富组件
- ⚡ HTTP 连接池机制
  - 复用连接减少握手开销
  - 自动管理连接生命周期
  - 显著提升响应速度
- 🎬 Web 控制台动画优化
  - 15+ 种流畅的过渡动画
  - 优化的主题切换效果
  - 增强的表单元素交互
  - 骨架屏加载状态
  - 平滑滚动和自定义滚动条

### 增强
- 🔒 安全性增强
  - 请求体大小限制（默认 10MB）
  - 自动添加安全响应头
    - X-Content-Type-Options: nosniff
    - X-Frame-Options: DENY
    - X-XSS-Protection: 1; mode=block
  - 严格的输入验证
- 🚀 性能优化
  - 连接池复用减少延迟
  - 优化的 CSS 过渡效果
  - 减少不必要的 DOM 操作
- 💅 UI/UX 改进
  - 统一的 CLI 输出风格
  - 更清晰的错误提示
  - 改进的菜单导航
  - 响应式动画效果

### 修复
- 🐛 修复 CLI 递归调用 bug
- 🐛 修复连接未正确关闭的问题
- 🐛 改进错误处理和异常捕获
- 🐛 修复流式响应的连接管理

### 变更
- 📝 完全重写 README.md，更清晰的结构
- 📦 添加 rich 依赖到 pyproject.toml
- 🔧 优化代码结构和模块化
